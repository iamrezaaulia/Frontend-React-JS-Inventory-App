## Requirements
1. <a href="https://nodejs.org/en/download/">Node Js</a>
2. Node_modules
3. React Native CLI

## How to run the app ?
1. Open app's directory in CMD or Terminal
2. Type `npm install`
3. prepare the emulator to run the application or you can use your Smartphone
4. Type ` react-native run-android `
